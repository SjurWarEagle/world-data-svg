# Usage
Load the module:
```typescript

@NgModule({
[...]
  imports: [
[...]
    WorldCountryDataLibModule
  ],
[...]

```
Use the component `world-country` and bind your data to it:
```typescript
      <world-country
        [worldData]="worldDataBasicStatic"
      ></world-country>
```
